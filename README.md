# Overview

This project analyzes employee attrition patterns using an HR dataset to identify factors associated with employee turnover. The analysis explores differences across departments, job roles, tenure, overtime, and compensation, and includes an exploratory logistic regression model.

# Key Findings

* Employees working overtime have significantly higher attrition rates and ~4.3x higher odds of leaving.

* Attrition is highest among early-tenure employees (0–2 years).

* Certain job roles, particularly sales-related positions, experience elevated turnover.

* Higher job satisfaction and longer tenure are associated with lower attrition.

# Methods

* Data cleaning and transformation in R

* Exploratory data analysis (EDA)

* Visualization using ggplot2

* Logistic regression modeling to estimate attrition odds
